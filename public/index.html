PDF Copy Protection // PDF-Kopierschutz

Prepare a PDF for distribution // Ein PDF für die Weitergabe an Dritte vorbereiten

  • Images in your PDF will be reduced to a single layer so that it is harder to copy objects. // Bilder in Deinem PDF werden auf eine Ebene reduziert, sodass einzelne Objekte schwerer zu kopieren sind.
  • All features expect printing will be deactivated. // Alle Funktionen außer dem Drucken werden deaktiviert.
  • Your PDF can have a maximum size of 20 MB. // Das PDF darf maximal 20 MB groß sein.
  • Important: // Wichtig: Always keep your original PDFs. // Behalte immer Deine originalen PDFs.
